`Maven` Installation Guide
`Maven` is a build automation tool used primarily for Java projects. To install `Maven` in ubuntu linux follow this guide.

### Pre-requisite:
- `Java`

### Step1:
Open `Terminal` and type this cmd:

```bash
sudo apt-get update
```
and press `ENTER` key.

### Step2:
Now type this cmd:

```bash
sudo apt-get install maven
```
and press `ENTER` key.
